# Release Notes - Version 1.5.0

**Release Date:** December 13, 2024

## 🎨 Major Visual Overhaul - Vintage Space Monster Theme

Version 1.5.0 introduces a complete visual transformation with a dramatic vintage sci-fi horror aesthetic inspired by classic space monster movies.

### New Visual Design
- **Retro Sci-Fi Styling**: Complete theme overhaul with high-contrast black/white/cream color palette
- **Dramatic Red Accents**: Pure red (#ff0000) highlights for alerts, buttons, and interactive elements
- **Vintage Typography**: New font system using Bebas Neue, Rubik Mono One, and Creepster fonts
- **Special Effects**: Scanline overlays, noise textures, and subtle animations for authentic retro monitor feel
- **Sci-Fi UI Elements**: Sharp corners, wide letter spacing, and uppercase text throughout

### Enhanced User Experience
- **Improved Legibility**: Increased letter spacing and removed glow effects for better readability
- **Centered Navigation**: Learn About and Create Warband buttons now centered for better visual balance
- **Compact Actions**: Reduced button padding on warband action buttons (Edit, Clone, Delete)
- **Thematic Messaging**: Updated empty states and alerts with sci-fi themed text

## 🔄 Warband Management Features

### Warband Duplication
- **Clone Warbands**: New ability to duplicate existing warbands for quick army variations
- **Confirmation Dialog**: Safe duplication with user confirmation to prevent accidents
- **Updated Terminology**: Changed "Duplicate" to "Clone" for better sci-fi theming
- **Error Handling**: Robust error handling with user-friendly notifications

### Weirdo Duplication
- **Clone Weirdos**: Duplicate individual weirdos within a warband
- **Smart State Management**: Proper handling of unsaved warbands and duplicate restrictions
- **Integration Testing**: Comprehensive test coverage for duplication workflows

## 📚 Learn About Integration

### README Content System
- **In-App Documentation**: New "Learn About Space Weirdos" feature with integrated README content
- **Dynamic Loading**: Fetches and displays project documentation within the application
- **Fallback Content**: Graceful handling when content cannot be loaded
- **Popup Interface**: Clean, accessible popup for viewing documentation

### Content Management
- **Caching System**: Efficient content caching for improved performance
- **Error Recovery**: Automatic fallback to cached content when network requests fail
- **Accessibility**: Full keyboard navigation and screen reader support
